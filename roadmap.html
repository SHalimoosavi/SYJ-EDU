Roadmap

Built in the open, versioned like the rest of the project.

This roadmap reflects reality, not marketing. Items move from planned to in progress to shipped only when their projects and lessons are actually merged.

1 Shipped · v1.0

Core learning tracks

AI Engineering, Backend Engineering, Database Engineering, Frontend Engineering, Cybersecurity, and OSINT tracks published with full curricula and starter repositories.

2 Shipped · v1.1

Automation, SaaS & CRM Development tracks

Three additional tracks published, extending the curriculum to nine total learning paths.

3 In progress

Industry project library

Domain-specific builds across healthcare, agriculture, logistics, transportation, cloud kitchen, restaurant, retail, education, finance, real estate, manufacturing, customer support, and small business.

4 Planned

Community mentorship program

Structured pairing and code review for contributors working through the advanced tracks (AI Engineering, Cybersecurity, SaaS Development).

5 Planned

Project completion badges

Verifiable, non-commercial completion badges tied to merged pull requests against the official project repositories — not multiple-choice quizzes.

6 Planned

Translated curricula

Community-contributed translations of core tracks, starting with the highest-demand languages requested via GitHub Discussions.

Want to influence what ships next?

Roadmap priorities are shaped directly by GitHub Issues and Discussions — not a closed internal backlog.
