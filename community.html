Community

Learning in public, together.

SYJ EDUCATE has no forums to moderate and no Discord to babysit — the community lives directly on GitHub, where the curriculum itself is developed.

Code of conduct, in short

  • Be direct and honest in code review — vague praise doesn't help anyone learn.
  • No harassment, discrimination, or personal attacks, on any thread.
  • Assume good faith from beginners; everyone's first pull request is rough.
  • Keep security research (Cybersecurity and OSINT tracks) scoped to the provided lab environments only.
