Why SYJ EDUCATE

Built differently, on purpose

Production-grade only

No to-do apps. Every project mirrors a real system you'd ship to users, complete with tests and deployment steps.

Zero cost, forever

MIT-licensed and free to use, fork, and redistribute. No paywalled modules, no "pro tier" bait-and-switch.

Built by a solo founder, in public

Every track is derived from real products shipped under the SYJ brand — including the mistakes, rewrites, and fixes.

Runs on constrained hardware

Curriculum favors zero-native-compilation, single-file, and CDN-based approaches — the same constraints of building on Termux.

Open source philosophy

Curriculum as a public repository, not a product funnel

SYJ EDUCATE is released under the MIT License. Anyone can fork the curriculum, adapt it to their own cohort, or contribute a new project back to the main repository.

MIT License

Full commercial and personal reuse rights, with attribution appreciated but not required by law.

Open contributions

Pull requests are welcomed for new lessons, bug fixes, translations, and entirely new tracks.

Public issue tracker

Report unclear lessons, broken project starters, or outdated dependencies directly on GitHub Issues.

Transparent roadmap

Every planned track and feature is visible on the Roadmap page before it ships.
